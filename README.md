## 📌 Project Overview
This project tackles a classic computer vision challenge: distinguishing between highly similar visual textures and shapes, specifically Chihuahuas and blueberry muffins. The system utilizes both Classical Machine Learning (SVM) and Deep Learning (Convolutional Neural Networks) to classify images and evaluate performance metrics.

3. Run `CNN_1_Chihuahua_or_Muffin.ipynb` sequentially to load the dataset, train the CNN, and output the classification predictions.
